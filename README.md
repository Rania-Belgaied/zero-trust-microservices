# Zero-Trust Microservices Security Framework

Sécurisation d'une architecture microservices par une approche Zero-Trust
basée sur un système multi-agents.

## Stack technique
- Kubernetes (Minikube)
- Istio (mTLS)
- Keycloak (identités)
- OPA (autorisation)
- Falco (runtime security)
- Sealed Secrets (gestion des secrets)
- Prometheus + Grafana (observabilité)
- Python (agents LSA et CCA)
